🦞 OpenClaw

🔗 访问官网

📦 安装流程

  • 1安装 OpenClaw
    Windows:
    iwr -useb https://openclaw.ai/install.ps1 | iex
    或者
    npm install -g openclaw@latest
    macOS/Linux:
    curl -fsSL https://openclaw.ai/install.sh | bash
  • 2运行引导配置
    openclaw onboard --install-daemon
    配置认证、网关设置和可选频道
  • 3检查网关状态
    openclaw gateway status
    确认服务正在运行
  • 4打开控制台
    openclaw dashboard
  • 5配置频道（可选）
    openclaw channels login
    配对 WhatsApp/Telegram 等
☁️ 云端部署腾讯云/阿里云均可部署，个人选择2核2G配置即可
💻 本地部署Windows需要提前安装node.js、Git；直接在个人电脑运行，适合开发和测试使用
